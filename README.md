# C++ Practical Exam

## Description

This repository contains C++ practical programs based on Object-Oriented Programming concepts.

## Questions Covered

### 1. Student Class

* Created a `Student` class.
* Used private attributes:

  * name
  * rollNumber
* Used parameterized constructor.
* Created array of objects.

### 2. Car Class

* Created a `Car` class.
* Used encapsulation.
* Private attributes:

  * model
  * year
  * speed
* Used setter and getter functions.

### 3. Device Class

* Created base class `Device`.
* Derived classes:

  * Laptop
  * Smartphone
* Used abstraction by calling display methods.

### 4. Device Discount

* Added virtual function `calculateDiscount()`.
* Implemented function in derived classes.
* Used array of `Device` pointers.
* Demonstrated polymorphism.

### 5. Animal Abstract Class

* Created abstract class `Animal`.
* Used pure virtual functions:

  * makeSound()
  * move()
* Derived classes:

  * Lion
  * Fish
* Demonstrated polymorphism using object pointers.

## OOP Concepts Used

* Class and Object
* Constructor
* Encapsulation
* Inheritance
* Abstraction
* Polymorphism
* Virtual Function
* Pure Virtual Function
